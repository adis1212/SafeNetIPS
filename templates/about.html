About SafeNet IPS

SafeNet.ai is an AI-powered Intrusion Prevention System (IPS) built using Flask. It identifies and prevents a wide range of cyber threats in real-time, including:

  • Network-Based Attacks: DoS, DDoS
  • Application Attacks: SQL Injection, XSS, CSRF
  • Malware Attacks: Spyware, Keyloggers
  • Insider Threats: Unauthorized access, Data exfiltration
  • Email Threats: Phishing, Spoofing
  • Brute Force Attacks: Password cracking

This system uses a trained machine learning model and logs any malicious attempts for audit and monitoring.
